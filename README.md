# FeatureFlag
A micro-service for multi-treatment testing with various options for experiements

## SDKS
- UI/Frontend SDK for quickly adding A/B/n testing
  - React/Next.js support
- C# & Go SDK for server-side applications
- Cloud-agnostic adaptors for self-hosting & cloud hosting

## Features
- Browser-based overrides (localStorage, cookies)
- Configurable default (in case of outage)
- Customizable ramp-up and auto-ramp down
